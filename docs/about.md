# About TechLaboratory

We stands for creating and delivering free and open source projects. Launched on September 2010 by [Dipu Raj](http://dipu.me), we are still keep supporting the open-source world.

## Our Mission

TechLaboratory is dedicated to creating beautiful, modern, and flexible React components and jQuery plugins with Bootstrap support. All our projects are open-source and available for free.

## Our Projects

We have developed amazing React components and jQuery plugins that help developers build better web applications:

- **React Smart Tab** - The awesome react tab component for ReactJS
- **jQuery Smart Wizard** - The awesome step wizard plugin for jQuery
- **jQuery Smart Cart** - jQuery Shopping Cart plugin with PayPal payment support
- **jQuery Smart Tab** - The flexible jQuery tab control plugin
- **create-jquery-plugin** - CLI for creating ready-to-start modern jQuery Plugins
